# Panopticon

> **[English Version / 英文版本](README_en.md)**

[Panopticon AI](https://panopticon-ai.com/) 是一个基于Web的军事仿真平台，兼容 OpenAI Gym。我们的项目是开源的，
代码面向军事建模与仿真研究人员开放。
我们希望推动军事AI研究的技术发展。

平台：[我们的应用](https://app.panopticon-ai.com/)。
