# 侧边栏菜单

## 文件选项

场景存储在 JSON 文件中。您可以创建新场景或上传场景。使用铅笔图标编辑场景名称。

使用 Google 账户登录后，您可以将场景保存到云端或下载场景文件。

## 编辑场景

播放部分允许您将场景重置为开始、播放和暂停、步进和更改播放速度。

撤消按钮允许您撤消编辑。

## 录制

录制与场景不同，因为它们跟踪场景随时间的推移如何发展。录制按钮允许您保存录制回放。由于文件大小，应用程序将下载每个 10MB 的多个文件。切换场景时间步录制以减小文件大小。

## 任务

任务允许您为一组飞机创建预先规划的路线。目前支持打击任务和巡逻任务。

请注意，任务是为特定方创建的。

巡逻任务需要参考点，创建一个多边形。从顶部栏添加参考点以创建任务颜色的多边形。选择几个单位和参考点后，飞机将进行巡逻。

打击任务的工作方式类似，但需要一个目标。

请注意，飞机必须装载武器才能执行任务。

## 图层

可以循环切换不同的底图。

使用图层按钮打开和关闭某些图层。

# 顶部栏菜单

## 编辑按钮

顶部栏主要用于添加单位。支持飞机、舰船、SAM 和参考点。还支持预设的空军基地位置。

要创建自定义空军基地，请右键单击地图。

数据库按钮允许您自定义单位数据。请勿上传机密信息。

消息日志记录事件，允许您调试场景中发生的情况。

## 盟友和条令

下拉菜单允许您创建各方。单击铅笔图标控制盟友和敌人。

条令菜单允许您选择方的预期行为。

# 单位类型

## 飞机

单击单位会显示一个包含可编辑特征的菜单。单击铅笔编辑此单位的特征。注意探测范围在决定条令中的重要性。

单击操作如下：

- **规划航线**：沿航线方向行进
- **手动攻击**：打开武器菜单
- **RTB**：返回原始基地
- **复制**：创建一个重复单位。
- **编辑位置**：将单位移动到不同位置。

## 武器菜单

武器菜单允许您添加武器和发射导弹。使用火箭图标手动发射导弹。

也可以在条令菜单中自动发射导弹。

## SAM

SAM 具有坐标位置、高度、射程、方和武器数量。对于 SAM，用户可以编辑其名称、类型、射程和武器数量。

## 舰船（开发中）

舰船具有坐标位置、速度、航向、飞机数量和舰船燃料。

对于舰船，用户可以编辑其名称、类型、速度、武器数量、当前燃料和射程。

用户可以使用起飞飞机按钮从舰船上起飞飞机。

## 基地

空军基地主要用于停放飞机。空军基地具有坐标位置、高度、方和飞机数量。

对于空军基地，用户可以编辑其名称。

操作允许您向基地添加飞机。
