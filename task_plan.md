## 已做决策
| 决策 | 理由 |
|------|------|
| 使用 react-i18next | React 生态成熟的国际化解决方案 |
| 支持中英文 | 根据用户需求，支持中文和英文两种语言 |
| 语言切换按钮放在 Toolbar 右侧 | 靠近 LoginLogout 按钮，符合用户习惯 |
| 使用 localStorage 持久化语言选择 | 用户刷新页面后保持语言偏好 |
